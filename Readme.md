## 🚀 Highlight Project – Top 10 Largest Economies by GDP

In Course 04, I developed a Python script capable of:

- Extracting data from the web  
- Parsing HTML tables using `pandas`  
- Cleaning and transforming data  
- Identifying the 10 largest economies in the world by GDP (IMF data)  
- Converting values to billions of USD  
- Rounding results to two decimal places  

This project reinforced important skills such as:

- Web scraping  
- DataFrame manipulation  
- Data cleaning and transformation  
- Exception handling
